## 打印到控制台
### 打印字符串
#### 输出方式
Rust 输出文字的方式主要有两种：
1. `println!()` 
2. `print!()`。
区别在于:
- `println!`在输出的最后附加输出一个换行符
- `println` 允许打印空的字符串到新的一行，`print!`不允许 

#### println!
```rs
fn main(){
    println!(); // prints just a newline
    println!("hello there!");
    println!("format {} arguments {1}", "some", 123);
    println!("format {{}}");
}
```
总结：
- 使用 `{}` 作占位符
- 可以使用`{number}`指定变量位置
- 使用`{{}}`转义显示字符串`{}`
